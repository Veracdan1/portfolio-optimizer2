# Portfolio Optimizer (Project 2 – Path A)

This project was built for MGMT 49000 – AI Finance Application.

## Overview
This application is a portfolio optimization tool that allows users to input 5–15 stock tickers and generate an optimal portfolio based on different objectives and constraints. It retrieves historical price data from Yahoo Finance and calculates expected returns, risk, and optimal weights.

## Features
- Accepts 5–15 stock tickers
- Retrieves historical price data from Yahoo Finance
- Calculates expected returns and covariance matrix
- Supports optimization using:
  - Maximum Sharpe Ratio
  - Minimum Variance
  - Target Return
- Displays:
  - Optimal portfolio weights
  - Expected return
  - Expected risk (volatility)
- Allows user-adjustable constraints:
  - Max weight per stock
- Allows comparison of risk preferences
- Displays efficient frontier

## Data Source
- Historical stock price data is retrieved using the `yfinance` library from Yahoo Finance.

## Methodology
- Expected returns are calculated using historical average returns and annualized.
- Risk is measured using the covariance matrix.
- Portfolio optimization is performed using constrained optimization (SciPy).
- The efficient frontier is generated by solving for portfolios across a range of target returns.

## Limitations
- Historical returns may not predict future performance.
- Results are sensitive to input data and time period.
- Covariance relationships may change over time.
- Yahoo Finance data is used for convenience and may not be institution-grade.

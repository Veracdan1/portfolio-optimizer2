This project followed the DRIVER framework to build a portfolio optimization application. In the Discover &amp; Define stage, I identified the goal of creating a tool that accepts multiple stock tickers, retrieves historical data, and allows users to explore how different objectives and constraints affect portfolio outcomes. In Represent, I structured the solution into key components: data retrieval from Yahoo Finance, return and covariance calculations, optimization logic, and a user interface with adjustable inputs. In Implement, I used Python with Streamlit, yfinance, NumPy, and SciPy to build an interactive app that calculates expected returns, risk, and optimal weights under different objectives such as max Sharpe and minimum variance. In Validate, I tested the app using multiple ticker sets and verified that results were consistent with financial intuition, such as weights summing to one and risk increasing with higher return targets. In Evolve, I improved the app by adding constraints like maximum weight per stock and enhancing the efficient frontier visualization. Finally, in Reflect, I found that AI tools were helpful for structuring and debugging the code, but I needed to verify all financial logic and outputs to ensure accuracy and interpretability.
